# Color Game Component Based

#### 1. Easy mode
*3 cards, no time limit.
#### 2. Hard mode
*6 cards, no time limit.
#### 3. Nightmare mode
*6 cards, 5 seconds time limit.

#### Demo Gif
![](demo.gif)

## Important
1. git clone this repository.
2. Run your terminal.
3. cd to color-game-component-based.
4. Run these commands in order:
  * `npm install`
  * `npm run build`
  * `npm run watch`
